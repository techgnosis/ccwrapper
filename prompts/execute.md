Use `br ready` to find and work the next epic. Do not work more than one epic. Do not stop until the epic is completed. When you are done with the epic, make sure to update `br`. If a new agent comes in and runs `br ready` then it should have a discrete, unstarted epic to work on.

Always commit the code when you are done with an epic. Try and push but there might not be a remote and that is OK.
